1) Prompt

Rules for this demo (always visible)
  • Crop: corn (maize) only.
  • Location: 5-digit county GEOID, or county + state, or latitude/longitude.
  • Planting date: optional; default county planting date is used if omitted.
  • Year: optional; if you provide dates without a year, the demo assumes the current year unless you specify a year.
  • Current date: the start date for the next 14-day irrigation demand window. If omitted, it defaults to the local date at the simulation location (only for the current year). For a past year, you must provide a current date in that year.
  • Forecast limit: current date cannot be in the future (we need the next 14 days from that date). Past dates are simulated using historical weather (archive).
  • MAD (Maximum Allowable Depletion, %): optional; default is 50%.
