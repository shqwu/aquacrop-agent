AquaCrop Agent Demo

A lightweight web demo: prompt → FastAPI backend → AquaCrop run → JSON + summary table + irrigation plot.

Notes: This demo currently supports corn (maize) only. Location can be county GEOID, county+state, or lat/lon. Dates without a year default to the current year unless you specify a year.

1) Prompt

Ready.
Example prompts:
  • Run maize in Mitchell County, GA, planted January 5 2026, irrigated
  • 31.213 -84.216 irrigated, planting Jan 5, year 2026
  • Mitchell County GA, current date Feb 28, MAD 45%
  • GEOID 13205 rainfed planting 1/10 (if ambiguous, the agent will ask)
Need more info:
Tip: the backend is currently stateless. If you answer a clarification, you can paste a clear follow-up prompt, e.g. Planting date: January 10, 2026 or Option 1.
